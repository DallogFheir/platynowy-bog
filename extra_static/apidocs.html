API reference

items

/api/items

retrieves an array of all items

/api/items/<id>

retrieves an item by its ID

An item object contains the following properties:

  • id : integer
  • name : string
  • quote : string
  • type : string – possible values:
    • "active"
    • "passive"
  • quality : integer
  • pool : object – pools where the item can be found; possible properties:
    • normal : array[string] – array of pools in non-Greed(ier) mode
    • greedMode : array[string] – array of pools in Greed(ier) mode
    • boss : array[string] – array of bosses which drop the item
    • miniBoss : array[string] – array of mini-bosses which drop the item
    • chests : array[string] – array of chest types which can contain the item
    • beggars : array[string] – array of beggars which can pay out with the item
    • machines : array[string] – array of machines which can pay out with the item
    • obstacles : array[string] – array of obstacles which can drop the item after being destroyed
    • startingItem : array[string] – array of characters for whom the item is a starting item
    • other : array[string] – array of other possible ways to find the item; possible values:
      • "Baby Shop"
      • "other"
  • optional transformation : array[string] – array of transformations (if any)
  • optional rechargeTime : object – if the item is active, an object representing its recharge time; properties:
    • amount : integer | string – integer represeting the amount of rooms/seconds, or string equal to:
      • "one time use"
      • "unlimited"
      • "other"
    • optional unit : string – possible values:
      • "rooms"
      • "s"
