API reference

items ¶

GET /api/items

retrieves an array of all items – example

GET /api/items/<id>

retrieves an item by its ID – example

An item object contains the following properties:

  • id : integer
  • name : string
  • optional quote : string
  • type : string – possible values:
    • "active"
    • "passive"
  • optional quality : integer
  • pool : object – pools where the item can be found; possible properties:
    • normal : array[string] – array of pools in non–Greed(ier) mode
    • greedMode : array[string] – array of pools in Greed(ier) mode
    • boss : array[string] – array of bosses which drop the item (including the special Bumbino pool)
    • miniBoss : array[string] – array of mini–bosses which drop the item
    • chests : array[string] – array of chest types which can contain the item
    • beggars : array[string] – array of beggars which can pay out with the item
    • machines : array[string] – array of machines which can pay out with the item
    • obstacles : array[string] – array of obstacles which can drop the item after being destroyed
    • startingItem : array[string] – array of characters for whom the item is a starting item
    • other : array[string] – array of other possible ways to find the item; possible values:
      • "Baby Shop"
      • "other"
  • optional transformation : array[string] – array of transformations (if any)
  • optional rechargeTime : object – if the item is active, an object representing its recharge time; properties:
    • amount : integer | string – integer represeting the amount of rooms/seconds, or string equal to:
      • "one time use"
      • "unlimited"
      • "other"
    • optional unit : string – possible values:
      • "rooms"
      • "s"
  • optional unlock : object – object representing how to unlock the trinket; properties:
    • method : string – possible values:
      • "boss"
      • "challenge"
      • "Donation Machine"
      • "Greed Donation Machine"
      • "other"
    • optional boss : string – if the method of unlocking is "boss"; boss to defeat
    • optional character : string – if the method of unlocking is "boss"; character with which the boss needs to be defeated
    • optional numberOfTimes : integer – if the method of unlocking is "boss"; how many times the boss needs to be defeated
    • optional mode : string – if the method of unlocking is "boss"; the only possible value is "Hard Mode" if the boss needs to be defeated in Hard Mode
    • optional challengeNumber : integer – if the method of unlocking is "challenge"; the number of the challenge
    • optional challengeName : string – if the method of unlocking is "challenge"; the name of the challenge
    • optional amount : integer – if the method of unlocking is "Donation Machine" or "Greed Donation Machine"; the amount of money to donate to the machine
    • optional other : string – if the method of unlocking is "other"; the way to unlock the item

trinkets ¶

GET /api/trinkets

retrieves an array of all trinkets – example

GET /api/trinkets/<id>

retrieves a trinket by its ID – example

A trinket object contains the following properties:

  • id : integer
  • name : string
  • quote : string
  • optional setDrop : array[string] – array of set ways to obtain the trinket
  • optional unlock : object – object representing how to unlock the trinket; properties:
    • method : string – possible values:
      • "boss"
      • "challenge"
      • "other"
    • optional boss : string | array[string] – if the method of unlocking is "boss"; boss or bosses to defeat
    • optional character : string – if the method of unlocking is "boss"; character with which the boss needs to be defeated
    • optional challengeNumber : integer – if the method of unlocking is "challenge"; the number of the challenge
    • optional challengeName : string – if the method of unlocking is "challenge"; the name of the challenge
    • optional other : string – if the method of unlocking is "other"; the way to unlock the trinket

pickups ¶

GET /api/pickups

retrieves an array of all pickups – example

GET /api/pickups/<groupId>/[<id>]

retrieves a pickup by its group ID and (optionally) its ID – example

A pickup object contains the following properties:

  • groupId : integer
  • optional id : integer
  • name : string
  • type : string – possible values:
    • "heart"
    • "coin"
    • "key"
    • "battery"
    • "chest"
    • "sack"
  • optional unlock : object – object representing how to unlock the pickup; properties:
    • method : string – possible values:
      • "boss"
      • "challenge"
      • "other"
    • optional boss : string – if the method of unlocking is "boss"; boss to defeat
    • optional character : string – if the method of unlocking is "boss"; character with which the boss needs to be defeated
    • optional challengeNumber : integer – if the method of unlocking is "challenge"; the number of the challenge
    • optional challengeName : string – if the method of unlocking is "challenge"; the name of the challenge
    • optional other : string – if the method of unlocking is "other"; the way to unlock the trinket

cards/runes ¶

GET /api/cards-runes

retrieves an array of all cards/runes – example

GET /api/cards-runes/<id>

retrieves a card/rune by its ID – example

A card/rune object contains the following properties:

  • id : integer
  • name : string
  • quote : string
  • type : string – depends on the card's/rune's appearance on the ground; possible values:
    • "major arcana card"
    • "reversed major arcana card"
    • "playing card"
    • "Magic: The Gathering card"
    • "credit card"
    • "Card Against Humanity"
    • "chance card"
    • "Holy Card"
    • "Uno card"
    • "left-pointing rune"
    • "right-pointing rune"
    • "Black Rune"
    • "Rune Shard"
    • "Soul Stone"
  • optional unlock : object – object representing how to unlock the card/rune; properties:
    • method : string – possible values:
      • "boss"
      • "challenge"
      • "other"
    • optional boss : string | array[string] – if the method of unlocking is "boss"; boss or bosses to defeat
    • optional character : string – if the method of unlocking is "boss"; character with which the boss needs to be defeated
    • optional challengeNumber : integer – if the method of unlocking is "challenge"; the number of the challenge
    • optional challengeName : string – if the method of unlocking is "challenge"; the name of the challenge
    • optional other : string – if the method of unlocking is "other"; the way to unlock the trinket

transformations ¶

GET /api/transformations

retrieves an array of all transformations – example

GET /api/transformations/<id>

retrieves a transformation by its ID (Super Bum has the ID of 14) – example

A transformation object contains the following properties:

  • optional id : integer – Super Bum has no ID, but can be retrieved by the ID of 14
  • name : string
  • optional items : array[integer] – array of the IDs of items that contribute to the transformation
  • optional trinkets : array[integer] – array of the IDs of trinkets that contribute to the transformation
  • optional pills : array[string] – array of the names of pills that contribute to the transformation
